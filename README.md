# Cosmos-Possible-Worlds


  Este projeto foi realizado no âmbito da disciplina de Multimédia, pertencente ao 
Mestrado em Engenharia Informática da Faculdade de Ciências da Universidade de Lisboa. 

  Na temática aqui desenvolvida, centrámos a nossa atenção na elaboração de uma 
aplicação multimédia interativa com visualização e navegação de vídeo, tendo especial 
atenção a aspetos que contribuem para a sensação de presença e imersão.

  A nossa finalidade principal foi a obtenção de uma aplicação multimédia com a 
possibilidade de navegação no vídeo “Cosmos Possible Worlds Ladder to the Stars” através de 
texto e diversas imagens dispostas por ordem cronológica, e ainda o acesso a informação 
adicional externa sobre determinados temas apresentados nesse mesmo vídeo. 

  Uma inspiração para o desenvolvimento do nosso projeto foi “A História do Pi em 
Hipervídeo”. Este livro didático foi convertido em hipertexto, respeitando a sua estrutura 
hierárquica, e posteriormente enriquecido com links que envolvem texto, imagem, vídeo e 
aplicações, permitindo captar as relações entre eles e ilustrar ou complementar as 
informações veiculadas por cada um.

  Este projeto está alojado no seguinte website: https://cosmos-projeto-multimedia.herokuapp.com/
